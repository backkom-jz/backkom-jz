# Day 4：Web API 基础

## 复习知识点

Go：

- `net/http`
- Handler
- 路由处理
- 请求参数获取
- JSON 请求体解析
- JSON 响应
- HTTP 状态码

PHP：

- PHP Web 请求生命周期
- `$_GET`、`$_POST`、`$_SERVER`
- 请求参数处理
- JSON 输入读取
- JSON 响应输出
- 路由、控制器基础
- Laravel/ThinkPHP 基础路由
