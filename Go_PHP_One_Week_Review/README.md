# Go/PHP 一周复习目录

## 复习目标

用 7 天把 Go 和 PHP 后端开发的核心基础重新梳理一遍，重点不是背语法，而是能写出代码、能解释差异、能完成一个小型 API 项目。

## 每天节奏

- Go 复习：1.5 小时
- PHP 复习：1.5 小时
- 作业编码：1 小时
- 当天总结：30 分钟
