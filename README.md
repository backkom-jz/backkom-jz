## 复习顺序

1. 先读总手册，建立面试回答框架。
2. MySQL、Redis、RabbitMQ 每天按题库口述一遍。
3. PHP 框架源码按“入口、容器、路由、中间件、ORM、队列”准备。
4. 准备 3 个项目深挖案例：高并发、复杂业务、稳定性治理。
5. 最后做 2 轮模拟面试，重点训练项目表达和场景题取舍。
